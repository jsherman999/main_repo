# Example Screenshots

Place your screenshot files here for testing the multi-agent documentation system.

## Supported Formats

- PNG (.png)
- JPEG (.jpg, .jpeg)
- WebP (.webp)

## Example Usage

```bash
node src/main.js examples/your-screenshot.png \
  --app-name "Your Application" \
  --description "Brief description of what the screenshot shows" \
  --vendor "Vendor Name" \
  --links "https://example.com"
```

## Tips for Best Results

1. **High Resolution**: Use high-quality screenshots (1920x1080 or higher recommended)
2. **Clear UI**: Ensure UI elements are visible and not obscured
3. **Full Interface**: Capture the complete interface or relevant section
4. **Good Context**: Provide detailed description in the --description parameter
5. **File Size**: Keep files under 10MB for optimal processing
